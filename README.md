# Habit Tracker (Pelacak Kebiasaan)

Aplikasi **Habit Tracker (Pelacak Kebiasaan)** berbasis web untuk membantu pengguna melacak perkembangan kebiasaan setiap minggu. Pengguna dapat menambahkan kebiasaan, mencentang aktivitas setiap hari, melihat progress bar harian, serta statistik pencapaian mingguan.
Dibuat untuk memenuhi tugas **UTS Pemrograman Front-End**.
